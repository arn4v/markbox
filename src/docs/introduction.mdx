Bookmarky is an API-first bookmarking tool with a focus on design. It's built with React, TailwindCSS, Next.js and GraphQL.
It was born out of a sense of dissatisfaction with design of existing tools like [Pinboard](https://pinboard.in).

## Tech Stack

- React
- Next.js
- GraphQL
- MySQL
- Prisma 2
