Bookmarky uses **bookmarklets** for saving tabs in the browser.

If you're using Chrome/Edge/Opera, Firefox or Safari you can drag the following link to your bookmarks bar to get started.

- <a href="javascript:q=location.href;if(document.getSelection){d=document.getSelection();}else{d='';};p=document.title;void(open('https://bookmarky.io/create?url='+encodeURIComponent(q)+'&popup=true'+'&title='+encodeURIComponent(p),'Bookmarky','toolbar=no,scrollbars=yes,width=750,height=700'));">
  	Bookmarky
  </a>

or save the following code as a bookmark in browser:

```
javascript:q=location.href;if(document.getSelection){d=document.getSelection();}else{d='';};p=document.title;void(open('https://bookmarky.io/create?url='+encodeURIComponent(q)+'&popup=true'+'&title='+encodeURIComponent(p),'Bookmarky','toolbar=no,scrollbars=yes,width=750,height=700'));
```

**Side note**: You can edit the title of the saved bookmark like you would with any other bookmark in the browser. Right click, press edit and change the title.
